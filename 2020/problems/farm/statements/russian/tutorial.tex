\begin{tutorial}{Зимовье}

Простая задача на ветвления. Пример решения на языке программирования~C++ приведен ниже:
\begin{verbatim}
#include<iostream>
using namespace std;

void main(){
   int k,l,m;
   cin >> k >> l >> m;
   if (k*l<=m)
      cout << "YES";
   else
      cout << "NO " << k * l - m;
}
\end{verbatim}
